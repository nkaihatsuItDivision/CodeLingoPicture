📊 printf フォーマット指定子

美しい出力を作るための書式指定ガイド

💡 printf のフォーマット指定とは？
%から始まる特殊な記号で、変数の値をどのような形式で表示するかを指定する機能です。
桁揃え、小数点制御、文字列フォーマットなど、プロフェッショナルな出力が可能になります。
%s
文字列（String）
String 型の値を表示
📝 基本例:
String name = "田中";
System.out.printf("名前: %s", name);
名前: 田中
%d
整数（Decimal）
int, long 型の値を10進数で表示
📝 基本例:
int age = 25;
System.out.printf("年齢: %d歳", age);
年齢: 25歳
%f
小数（Float）
double, float 型の値を小数で表示
📝 基本例:
double height = 170.5;
System.out.printf("身長: %.1fcm", height);
身長: 170.5cm
%c
文字（Character）
char 型の値を1文字で表示
📝 基本例:
char grade = 'A';
System.out.printf("評価: %c", grade);
評価: A
🎯 高度な使い方：桁数指定と桁揃え
👀 フォーマット指定の動作確認
printf("%5d", 123)
→
__123
printf("%-5d", 123)
→
123__
printf("%05d", 123)
→
00123
printf("%.2f", 3.14159)
→
3.14
printf("%,d", 1234567)
→
1,234,567

※ __は半角スペースを表示

📏 桁数指定
%8d // 8桁右揃え
%-8d // 8桁左揃え
%08d // 8桁0埋め
🔢 小数点制御
%.2f // 小数点以下2桁
%8.2f // 8桁幅、小数2桁
%.0f // 整数部分のみ
💰 特殊機能
%,d // 3桁区切り
%+d // 符号を常に表示
%x // 16進数表示
💡 実用的な Tips
  • %,d を使うと数値に3桁区切りのカンマが自動で入る
  • %.2f で金額を小数点以下2桁で表示できる
  • %8s や %-8s で表形式のデータを美しく整列
  • %n は改行文字（\nと同じ）でクロスプラットフォーム対応
  • 複数の値は順番に printf("名前:%s 年齢:%d", name, age) で指定
