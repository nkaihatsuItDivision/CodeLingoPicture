📚 レッスン15 学生管理システムの設計例

🗃️ データ構造設計

学生情報の管理方法：
• names[] : 学生名を格納する配列
• scores[] : 点数を格納する配列
• count : 現在登録されている学生数
• MAX_STUDENTS : 最大登録可能数（定数）

配列のインデックス対応：
names[0] ↔ scores[0] ← 1番目の学生
names[1] ↔ scores[1] ← 2番目の学生

⚙️ メソッド設計

showMenu()
• 目的: メニューを表示する
• 引数: なし / 戻り値: なし

addStudent()
• 目的: 新しい学生を登録する
• 処理: 名前と点数を入力、配列に追加

showStudents()
• 目的: 全学生の情報を表示する
• 処理: 配列をループして情報を出力

🔄 処理フロー設計

メインループ：
1. while(true) で無限ループ
2. showMenu() でメニュー表示
3. scanner.nextInt() で選択受付
4. switch文 で処理分岐
5. 各機能の実行
6. 結果表示後、ループ継続

エラーハンドリング：
• try-catch で例外処理
• 不正入力時の再入力促進

🏗️ クラス設計

public class StudentManager { // データ（フィールド） private static String[] names; private static int[] scores; private static int count; private static Scanner scanner; // 機能（メソッド） public static void main(String[] args) public static void showMenu() public static void addStudent() public static void showStudents() public static void deleteStudent() public static void showStatistics() }
