⚖️ 比較演算子の動作説明
==
等しい
10 == 10 true
10 == 3 false
!=
等しくない
10 != 3 true
10 != 10 false
>
より大きい
10 > 3 true
3 > 10 false
<
より小さい
3 < 10 true
10 < 3 false
>=
以上
10 >= 10 true
10 >= 3 true
3 >= 10 false
<=
以下
3 <= 10 true
10 <= 10 true
10 <= 3 false
🎯 実用的な例：成績判定プログラム
int score = 85; // 合格判定（60点以上） boolean isPass = (score >= 60); // 優秀判定（80点以上） boolean isExcellent = (score >= 80); // 満点判定 boolean isPerfect = (score == 100); System.out.println("点数: " + score); System.out.println("合格? " + isPass); System.out.println("優秀? " + isExcellent); System.out.println("満点? " + isPerfect);
🖥️ 実行結果
点数: 85
合格? true
優秀? true
満点? false
🚨 重要な注意点
「等しい」は == （イコール2個） を使います！
= （イコール1個） は変数への代入に使うので混同しないように！

比較演算子の結果は必ず boolean型（true または false） になります。
