演算子の優先順位完全版（高い順）
優先順位 演算子 種類 例
1 ( ) 括弧 (a + b) * c
2 ++, --, ! 単項演算子 ++i, --j, !flag
3 *, /, % 乗除余 a * b, x / y, n % 2
4 +, - 加減 a + b, x - y
5 <, >, <=, >= 比較 a < b, x >= 10
6 ==, != 等価 a == b, flag != true
7 && 論理AND a > 0 && b < 10
8 || 論理OR x == 1 || y == 2
9 ? : 三項演算子 score >= 60 ? "合格" : "不合格"
10 =, +=, -= 代入 x = 5, count += 1
🔍 複雑な式の計算順序例
boolean result = 5 + 3 * 2 > 10 && true || false;
1
3 * 2 = 6 (乗算が最優先)
2
5 + 6 = 11 (次に加算)
3
11 > 10 = true (比較演算)
4
true && true = true (論理AND)
5
true || false = true (論理OR)
6
result = true (最後に代入)

• 可読性が向上します
• バグを防げます
• チーム開発で意図が明確になります
