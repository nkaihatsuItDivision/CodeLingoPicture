🔧 Git環境準備・インストール手順
1
Git公式サイトにアクセス
Gitの公式ウェブサイトから最新版のインストーラーをダウンロードします
https://git-scm.com/
2
Download for Windows をクリック
お使いのWindows環境に適した最新版が自動的に選択されます
3
インストーラーを実行
ダウンロードしたインストーラーファイルをダブルクリックして実行します
4
デフォルト設定でインストール
特別な理由がない限り、デフォルト設定のまま「Next」を連続でクリック
5
インストール完了
「Finish」をクリックしてインストールを完了します
インストール確認方法
コマンドプロンプトまたはPowerShellを開いて以下を実行：
git --version
正常にインストールされていれば、以下のような結果が表示されます：
git version 2.41.0.windows.1
重要な注意事項
インストール後は、コマンドプロンプトを再起動してからバージョン確認を行ってください。 既に開いていたコマンドプロンプトでは、パスが更新されていない可能性があります。
