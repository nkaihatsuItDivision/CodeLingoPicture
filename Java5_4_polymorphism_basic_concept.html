ポリモーフィズムの基本概念
日常生活の例：リモコンの電源ボタン
⚡
テレビリモコン
テレビの電源
ON/OFF
⚡
エアコンリモコン
エアコンの電源
ON/OFF
⚡
オーディオリモコン
オーディオの電源
ON/OFF
🔄 同じ操作（電源ボタンを押す）で、対象によって異なる動作が実行される
⬇️
プログラミングでの実現
// 同じメソッド呼び出し student1.displayInfo(); // 学部生の表示形式 student2.displayInfo(); // 大学院生の表示形式 // student1とstudent2の実際の型によって、 // 異なるdisplayInfo()が実行される
これがポリモーフィズムの本質！
同じメソッド呼び出しで、オブジェクトの種類に応じた
異なる動作を実現する仕組み
「同じインターフェースで異なる動作」
同じインターフェース
student.displayInfo()
→
異なる動作
学部生表示 or
大学院生表示
