for文の構文と各部分の役割
for (初期化; 条件; 更新) { 処理 }
初期化
条件
更新
for (int i = 1; i <= 5; i++) {
    System.out.println(i + "回目");
}
① 初期化
int i = 1
実行タイミング: 繰り返し開始前に1回だけ

役割:
• カウンター変数の宣言と初期値設定
• 繰り返し処理の起点を決める

例:
• int i = 1 (1から開始)
• int i = 0 (0から開始)
• int i = 10 (10から開始)
② 条件
i <= 5
実行タイミング: 毎回の繰り返し前にチェック

役割:
• 繰り返しを続けるかの判定
• trueなら続行、falseなら終了

例:
• i <= 5 (5以下の間続ける)
• i < 10 (10未満の間続ける)
• i != 0 (0でない間続ける)
③ 更新
i++
実行タイミング: 毎回の処理終了後

役割:
• カウンター変数の値を変更
• 次の繰り返しに向けた準備

例:
• i++ (1ずつ増加)
• i += 2 (2ずつ増加)
• i-- (1ずつ減少)

💡 重要なポイント

  • 初期化は1回だけ：最初に1回実行され、その後は実行されません
  • 条件は毎回チェック：処理前に必ず確認し、falseになると即座に終了
  • 更新は処理後：{ }内の処理が終わってから実行されます
  • 3つの部分は省略可能：ただし、初心者のうちは全て書くことを推奨
