🎯 type属性で input要素 の動作が変わる

同じ<input>でも、type属性を変えるだけで全く違う動作になります！

📝 type="text" - 一般的な文字入力
<input type="text" name="username" placeholder="ユーザー名を入力">
  • どんな文字でも入力可能
  • 名前、住所などに使用
  • 最も基本的な入力形式
実際の表示
📧 type="email" - メールアドレス専用
<input type="email" name="email" placeholder="例: yamada@example.com">
  • メールアドレス形式をチェック
  • スマホで@キーが出やすい
  • 送信前に自動検証
実際の表示
🚀 type="submit" - 送信ボタン
<input type="submit" value="送信">
  • フォームを送信するボタン
  • valueの内容がボタンに表示
  • クリックでフォーム送信実行
実際の表示

💡 実際に試してみよう！

上の入力欄に実際に文字を入力してみてください。特にtype="email"では、無効なメールアドレスを入力すると、ブラウザが教えてくれます。
