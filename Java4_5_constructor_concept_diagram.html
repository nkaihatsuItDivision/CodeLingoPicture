📚 コンストラクタの基本概念と実行タイミング
🎯 コンストラクタとは
オブジェクトが作成される時に自動的に実行される特別なメソッドです。
オブジェクトの初期化処理を担当し、安全で確実な状態でオブジェクトを使用開始できます。
🏷️
メソッド名 = クラス名
クラス名と全く同じ名前のメソッドです
🚫
戻り値の型なし
void も含めて戻り値の型を書きません
⚡
自動実行
new キーワードで自動的に呼び出されます
🔧
初期化処理
フィールドの初期化やセットアップを行います
⏰ コンストラクタの実行タイミング
1. new実行
new Student()
→
2. メモリ確保
オブジェクト用領域
→
3. コンストラクタ実行
初期化処理
→
4. オブジェクト完成
使用可能状態
💻 実際のコード例
// Studentクラスの定義
public class
Student {
    private String name;
    private int age;

    // コンストラクタ（メソッド名 = クラス名）
    public Student() {
        System.out.println("学生オブジェクトが作成されました");
        this.name = "";
        this.age = 18;
    }
}
1
コード実行
Student s = new Student();
→
2
コンストラクタ実行
初期化処理が自動実行
→
3
オブジェクト完成
使用可能状態
✅ 重要ポイント
• コンストラクタはオブジェクト作成時に必ず実行されます
• 初期화の安全性を保証する重要な仕組みです
• newキーワードと連動して自動的に動作します
• フィールドの初期値設定や必要なセットアップを確実に実行します
