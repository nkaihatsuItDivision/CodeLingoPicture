動的メソッド選択（Dynamic Method Selection）の仕組み
同じメソッド呼び出しでも、オブジェクトの実際の型に応じて適切なメソッドが自動選択される！

Student[] students = {
    new UndergraduateStudent("S001", "田中太郎", ...),  // 実際の型：UndergraduateStudent
    new GraduateStudent("G001", "佐藤花子", ...)        // 実際の型：GraduateStudent
};

for (Student student : students) {
    student.displayInfo();  // 同じ呼び出しでも異なるメソッドが実行される！
}
