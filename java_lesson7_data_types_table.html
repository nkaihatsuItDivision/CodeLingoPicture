Javaの8つの基本データ型（プリミティブ型）
4つのカテゴリに分類されています
カテゴリ データ型 サイズ 値の範囲 説明・用途
整数型
4種類		 byte 8bit -128 ～ 127 非常に小さな整数
ファイル処理、色の値
short 16bit -32,768 ～ 32,767 小さな整数
年数、座標値
int既習 32bit 約-21億 ～ 21億 一般的な整数
年齢、給与、点数
long 64bit 約-922京 ～ 922京 非常に大きな整数
時間ミリ秒、世界人口
浮動小数点型
2種類		 float 32bit 約7桁精度 単精度小数点数
ゲーム座標、メモリ節約
double 64bit 約15桁精度 倍精度小数点数
価格、距離、科学計算
文字型
1種類		 char 16bit Unicode文字 文字1個
'A', 'あ', '1', '@'
論理型
1種類		 boolean既習 1bit true / false 真偽値
フラグ、条件判定
整数型（4種類）
浮動小数点型（2種類）
文字型（1種類）
論理型（1種類）
💡 選び方のポイント
整数：基本的にはint型、大きな値はlong型
小数：基本的にはdouble型（高精度）
文字1個：char型、文字列：String型
真偽値：boolean型
