🛡️ privateによる保護

🔒 SafeStudentクラス 

public class SafeStudent {
    // ✅ 安全：全て private
    private String name;
    private int age; 
    private double gpa;
    
    // コンストラクタで検証
    public SafeStudent(String name, 
                    int age, double gpa) {
        this.name = (name != null) ? 
                  name : "未設定";
        this.age = (age >= 0) ? age : 0;
        this.gpa = (gpa >= 0.0 && 
                   gpa <= 4.0) ? gpa : 0.0;
    }
}

🔐 安全な更新メソッド 

// 検証付きメソッド
public void updateAge(int newAge) {
    if (newAge >= 0 && newAge <= 150) {
        this.age = newAge;
        System.out.println("更新成功");
    } else {
        System.out.println("無効な年齢");
    }
}

// 内部からのアクセスは可能
public void showInfo() {
    System.out.println("名前：" + 
                      this.name);
}

アクセス制御の仕組み

外部クラス
❌ アクセス不可
VS
クラス内部
✅ アクセス可能

privateフィールドはクラス内部からのみアクセス可能

✅ 安全な実行結果 

// 正常な更新
student.updateAge(21);  // "更新成功"

// 不正値は自動的に拒否
student.updateAge(-5);  // "無効な年齢"

// 直接アクセスは不可
// student.age = -5;  // コンパイルエラー
