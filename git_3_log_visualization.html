📖 Gitコミット履歴の確認方法
📋 基本的な履歴表示
git log
コミットの詳細情報を時系列順に表示。
コミットID、作成者、日時、メッセージが確認できます。
📊 統計情報付き表示
git log --stat
基本情報に加えて、各ファイルの変更行数も表示。
どのファイルがどの程度変更されたかが分かります。
📝 簡潔な履歴表示
git log --oneline
各コミットを1行で表示。
多数のコミットがある場合の概要確認に最適です。

💡 git log 活用のコツ
🎯 目的に応じた使い分け
概要確認：--oneline
詳細確認：--stat
基本確認：デフォルト
🔍 コミットIDの活用
短縮ID（a1b2c3d）で
特定のコミットを参照
過去バージョンへの復元も可能
📝 メッセージの重要性
分かりやすいメッセージで
履歴が読みやすくなる
後の開発作業が効率化
⏰ 開発進捗の可視化
いつ・誰が・何を変更したか
が一目で分かる
チーム開発での透明性確保
