なぜ int と String だけでは不十分なのか？

🧑 学生の年齢

20歳

小さな整数で十分
範囲: 0〜120程度

🌍 世界の人口

約80億人

とても大きな数値が必要
範囲: 8,000,000,000

💰 商品の価格

298.5円

小数点が必要
精密な計算が重要

🔤 文字コード

65 = 'A'

文字1個を数値で表現
Unicode対応が必要

現在のint型だけでは表現できない！
int age = 20; ✅ 年齢：問題なし
int worldPopulation = 8000000000; ❌ エラー！大きすぎる数値
int price = 298.5; ❌ エラー！小数点は使えない
int character = 'A'; ❌ 文字の扱いが不自然
解決策：多様なデータ型を使い分け！

様々な種類・サイズのデータに対応するため、Javaには8つの基本データ型が用意されています。

byte
short
int
long
float
double
char
boolean
