⚙️ Git初期設定・日本語対応設定
必須設定：ユーザー情報
1. ユーザー名の設定
$ git config --global user.name "あなたの名前"
設定例：
$ git config --global user.name "Taro Student"
2. メールアドレスの設定
$ git config --global user.email "あなたのメールアドレス"
設定例：
$ git config --global user.email "taro.student@example.com"
推奨設定：日本語対応
3. 日本語ファイル名の正しい表示
$ git config --global core.quotepath false
この設定により、日本語のファイル名やフォルダ名が正しく表示されるようになります
設定内容の確認方法
設定が正しく行われたか確認するコマンド：
$ git config --global --list
実行結果例：
user.name=Taro Student user.email=taro.student@example.com core.quotepath=false （その他の設定項目...）
🌟 この設定による効果
  • ✓
    コミット作成者情報が正しく記録される
  • ✓
    日本語ファイル名が文字化けしない
  • ✓
    チーム開発で作成者の識別が可能
  • ✓
    GitHub等のリモートサービスとの連携が円滑
  • ✓
    日本語プロジェクトでも安心して利用可能
