📖 Week 8: 表示技術の習得
🎯 主要技術要素
<!-- データ表示 -->
<p th:text="${student.name}"></p>
<p th:text="${student.email}"></p>

<!-- リスト表示 -->
<tr th:each="student : ${students}">
    <td th:text="${student.name}"></td>
    <td th:text="${student.age}"></td>
</tr>

<!-- 条件分岐 -->
<span th:if="${student.age >= 20}">成人</span>
📊 データフロー特徴
  • 単方向: Controller → View のみ
  • 読み取り専用: データの表示に特化
  • 静的処理: ユーザー入力なし
  • Model依存: Controller準備データを表示
📝 Week 9.2: フォーム技術の習得
🎯 主要技術要素
<!-- フォームバインディング -->
<form th:object="${studentData}">
    <input th:field="*{name}" />
    <input th:field="*{email}" />
    <input th:field="*{age}" />
</form>

<!-- データ受け取り -->
@PostMapping("/students")
public String create(
    @ModelAttribute StudentData data) {
    // 自動バインディング完了
}
📊 データフロー特徴
  • 双方向: Controller ⇄ View
  • 読み書き両用: 表示＋入力処理
  • 動的処理: ユーザー入力処理
  • 自動バインディング: フォーム⇄オブジェクト
