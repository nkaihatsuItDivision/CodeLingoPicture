🔍 括弧の効果：文字列連結 vs 算術演算
❌ 括弧なし（間違い）
System.out.println("a + b = " + a + b);
結果: "a + b = 103"
文字列連結として処理され、
"a + b = " + 10 + 3 = "a + b = 103"
となってしまいます
✅ 括弧あり（正解）
System.out.println("a + b = " + (a + b));
結果: "a + b = 13"
括弧内が先に計算され、
"a + b = " + (10 + 3) = "a + b = 13"
となります
📊 処理の流れを比較

❌ 括弧なしの場合

"a + b = " + 10
文字列連結
→
"a + b = 10" + 3
さらに文字列連結
→
"a + b = 103"
間違った結果

✅ 括弧ありの場合

(10 + 3)
括弧内を先に計算
→
"a + b = " + 13
文字列連結
→
"a + b = 13"
正しい結果
🎯 重要なポイント
数値の計算結果を文字列と連結したい場合は、
必ず括弧 ( ) を使って計算を先に実行させましょう！

括弧がないと、左から順番に処理されて文字列連結になってしまいます。
