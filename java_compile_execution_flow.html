📝 重要なポイント
1. 人間 → コンピューターへの翻訳
Javaプログラムは「人間が読める文法」で書いて、「コンピューターが理解できる形」に変換してから実行します。

2. Eclipseが自動化
この複雑な処理をEclipseが自動で行ってくれるので、私たちは「プログラムを書く」ことに集中できます。

3. エラーはコンパイル時に発見
文法の間違いなどは、コンパイル時（ステップ2）で発見されるので、実行前に修正できます。
