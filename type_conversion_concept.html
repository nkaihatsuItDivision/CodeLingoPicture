型変換の基本概念
🛡️ 暗黙的型変換（自動）
int number = 10;
double result = number;
↓ Javaが自動変換
// result = 10.0
  • Javaが自動で変換
  • 安全な変換のみ
  • 情報が失われない
  • 小さい型→大きい型
⚠️ 明示的キャスト（手動）
double decimal = 3.14;
int number = (int) decimal;
↓ プログラマーが指示
// number = 3 (小数点切り捨て)
  • プログラマーが明示
  • 情報損失の可能性
  • (型名)で変換指示
  • 大きい型→小さい型
🌟 日常生活での例え
暗黙的変換: 1km → 1000m（情報が増える、安全）
明示的変換: 1.5km → 1km（情報が減る、注意が必要）
