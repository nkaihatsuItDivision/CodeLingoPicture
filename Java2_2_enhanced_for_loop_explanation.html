🔧 構文比較：同じ処理を2つの方法で
❌ 従来のfor文（複雑）
// 学生名の一覧表示 ArrayList<String> students = getStudents(); for (int i = 0; i < students.size(); i++) { String name = students.get(i); System.out.println((i + 1) + ". " + name); } // 6行のコード、3つの変数管理
✅ 拡張for文（簡潔）
// 学生名の一覧表示 ArrayList<String> students = getStudents(); for (String name : students) { System.out.println(name); } // 3行のコード、1つの変数のみ

🌟 拡張for文を使う場面

✅ 適している場面:
• 全要素の走査
• インデックス不要
• 読み取り専用処理
• シンプルな処理
❌ 適していない場面:
• インデックスが必要
• 途中でbreak/continue
• 要素の変更・削除
• 複雑な制御
