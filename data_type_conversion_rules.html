データ型変換ルール
byte
1バイト
-128~127
short
2バイト
-32,768~32,767
int
4バイト
約-21億~21億
long
8バイト
約-922京~922京
float
4バイト
小数点7桁精度
double
8バイト
小数点15桁精度
→ 暗黙的変換（自動・安全）
← 明示的キャスト（手動・注意）
✅ 安全な変換（暗黙的）
int number = 100;
double result = number;
// result = 100.0（情報保持）
short s = 50;
int i = s;
// i = 50（情報保持）
⚠️ 注意が必要な変換（明示的）
double d = 3.14;
int i = (int) d;
// i = 3（小数点切り捨て）
long big = 300L;
byte small = (byte) big;
// 範囲チェック必要
