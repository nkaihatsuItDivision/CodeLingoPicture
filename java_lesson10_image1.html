論理演算子による条件判定の改善
❌ 従来の書き方（レッスン4まで）
if (age >= 18) { if (age < 65) { if (hasLicense) { System.out.println("運転できます"); } else { System.out.println("免許が必要です"); } } else { System.out.println("年齢制限があります"); } } else { System.out.println("年齢制限があります"); }
• ネストが深くて読みにくい
• 同じメッセージを複数箇所に記述
• 条件が増えると管理が困難
• 13行もの長いコード
✅ 論理演算子を使った改善版
if (age >= 18 && age < 65 && hasLicense) { System.out.println("運転できます"); } else { System.out.println("運転できません"); }
• 条件が1行で明確に表現
• ネストなしでスッキリ
• メンテナンスが簡単
• わずか5行の簡潔なコード
⬇️ 論理演算子の活用 ⬇️
主要な論理演算子
&&
かつ（AND）
すべての条件がtrueの時だけtrue
||
または（OR）
どれか一つでもtrueならtrue
!
ではない（NOT）
trueをfalseに、falseをtrueに反転
