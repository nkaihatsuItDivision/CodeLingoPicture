日常生活での判断 → プログラムでの条件分岐
🌟 日常生活での条件判断
☔
もし雨が降っているなら
傘を持って行く
🚦
もし信号が赤なら
止まる
⏰
もし時間があるなら
コーヒーを飲む
⬇️ これをプログラムで表現すると ⬇️
💻 プログラムでの条件分岐
// 雨の場合の判断
if (isRaining) {
    System.out.println("傘を持って行きましょう");
}
// 年齢の場合の判断
if (age >= 18) {
    System.out.println("成人です");
} else {
    System.out.println("未成年です");
}
🎯 同じ仕組み！
「条件を確認 → 結果に応じて行動」
