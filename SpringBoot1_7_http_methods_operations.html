🌐 HTTPメソッドと操作の対応
📖
GET
データの取得・読み込み専用
サーバーのデータを変更しない安全な操作
GET /api/students
GET /api/students/1

→ 学生一覧取得
→ ID=1の学生取得
➕
POST
新しいデータの作成・登録
リクエストボディにデータを含める
POST /api/students
Body: {"name":"田中太郎","age":20}

→ 新しい学生を登録
✏️
PUT
既存データの更新・編集
完全なデータで置き換え
PUT /api/students/1
Body: {"name":"田中次郎","age":21}

→ ID=1の学生情報を更新
🗑️
DELETE
データの削除・除去
指定されたリソースを削除
DELETE /api/students/1

→ ID=1の学生を削除
