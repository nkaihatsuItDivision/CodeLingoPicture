1
抽象クラスの定義
public abstract class Student { // 具体的実装（共通処理） public void displayInfo() { System.out.println("学生情報を表示"); } // 抽象メソッド（実装強制） public abstract void study(); public abstract String getStudentType(); }
⬇️
2
子クラスでの継承と実装
❌ 実装漏れ（コンパイルエラー）
public class UndergraduateStudent extends Student { // study()メソッドの実装を忘れた！ @Override public String getStudentType() { return "学部生"; } }
コンパイルエラー:
UndergraduateStudent is not abstract and does not override abstract method study() in Student
✅ 正しい実装（コンパイル成功）
public class UndergraduateStudent extends Student { @Override public void study() { System.out.println("学部レベルの学習"); } @Override public String getStudentType() { return "学部生"; } }
コンパイル成功:
全ての抽象メソッドが実装されています
