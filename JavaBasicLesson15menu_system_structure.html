// メニューシステムの基本構造
while (running) {
    showMenu();
    int choice = scanner.nextInt();

    switch (choice) {
        case 1:
            showStudentList();
            break;
        case 2:
            addStudent();
            break;
        case 0:
            running = false;
            break;
        default:
            System.out.println("無効な選択");
    }
}
🚀 switch-case文の利点
✅ 複数の選択肢を見やすく整理
✅ 実行速度が高速
✅ メニューシステムに最適
✅ break文で確実な制御
✅ default文でエラー対応
⚠️ 注意ポイント
🔸 break文を忘れるとフォールスルー
🔸 整数・文字・文字列のみ使用可
🔸 範囲指定（5以上など）は不可
🔸 複雑な条件はif文が適している
🔸 default文で予期しない値に対応
