🔍 LIKE演算子パターンマッチング完全ガイド
%
パーセント
任意の0文字以上の文字列
何にでもマッチ
_
アンダースコア
任意の1文字
必ず1文字にマッチ
1
前方一致検索
SELECT * FROM student
WHERE name LIKE '田%';
✅ マッチする例：
田中太郎
田村花子
田
田口一郎
❌ マッチしない例：
佐田太郎
山田美咲
2
後方一致検索
SELECT * FROM student
WHERE name LIKE '%子';
✅ マッチする例：
佐藤花子
田村美子
子
山本良子
❌ マッチしない例：
子供太郎
田中太郎
3
部分一致検索
SELECT * FROM student
WHERE department LIKE '%工学%';
✅ マッチする例：
情報工学部
工学部
機械工学科
電気工学専攻
❌ マッチしない例：
経済学部
文学部
4
1文字指定検索
SELECT * FROM student
WHERE student_id LIKE 'S00_';
✅ マッチする例：
S001
S002
S009
S00A
❌ マッチしない例：
S0001
S01
5
複合パターン検索
SELECT * FROM student
WHERE name LIKE '田_太郎';
✅ マッチする例：
田中太郎
田村太郎
田口太郎
❌ マッチしない例：
田太郎
田代一太郎
6
否定検索
SELECT * FROM student
WHERE department NOT LIKE '%経済%';
✅ マッチする例：
情報工学部
文学部
理学部
❌ マッチしない例：
経済学部
国際経済学部
