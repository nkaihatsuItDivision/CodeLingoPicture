student@PC: cd Week5StudentManagementSystem
student@PC: git branch
* main
📍 現在のブランチ状況
• * が付いているのが現在のブランチ
• main ブランチにいることを確認
• まだ他のブランチは作成されていません
student@PC: git branch -a
* main
remotes/origin/HEAD -> origin/main
remotes/origin/main
🌐 リモートブランチも含む表示
• -a オプション：すべてのブランチ表示
• remotes/origin/main：GitHubのmainブランチ
• ローカルとリモートが同期されている状態
