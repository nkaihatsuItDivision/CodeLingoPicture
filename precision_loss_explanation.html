型変換による精度損失
変換前
3.14159
double型
→
(int)
変換後
3
int型
0.14159 消失
⚠️ 精度損失が起こる理由
  • int型は整数のみ保存可能（小数点以下は保存できない）
  • 小数点以下は切り捨てられる（四捨五入ではない）
  • 一度失われた情報は復元できない
  • 大きな数値では桁あふれの可能性もある
様々な精度損失の例
double d1 = 123.456;
int i1 = (int) d1;
123.456 → 123
double d2 = -45.89;
int i2 = (int) d2;
-45.89 → -45
double d3 = 999.99;
int i3 = (int) d3;
999.99 → 999
精度損失を確認する方法
double original = 123.456;
int converted = (int) original;
double lost = original - converted;

System.out.println("元の値: " + original);
System.out.println("変換後: " + converted);
System.out.println("失われた値: " + lost);
実行結果:
元の値: 123.456
変換後: 123
失われた値: 0.456
