🔍 charAt() メソッドの使い方

String product = "iPhone15";
product.charAt(0) → 'i' (最初の文字)
product.charAt(6) → '1'
product.charAt(7) → '5' (最後の文字)
product.charAt(product.length() - 1) → '5' (最後の文字を取得する安全な方法)

💡 実践的な使い方

String name = "田中太郎";
char firstName = name.charAt(0); // '田'
char lastName = name.charAt(name.length() - 1); // '郎'

// 一文字ずつ表示
for (int i = 0; i < name.length(); i++) {
    System.out.println(i + "番目: " + name.charAt(i));
}
🎯 この概念を理解すれば、文字列の任意の位置にアクセスできるようになります！
