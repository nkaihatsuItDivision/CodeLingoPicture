📊 演算子の優先順位説明
🧮 数学と同じ！計算の順序があります
2 + 3 × 4 = 3 × 4 → 2 + 12 = 14
掛け算が先に計算されるのと同じように、
プログラミングでも演算子に優先順位があります！
🏆 演算子の優先順位（高い順）
1
( )
括弧 - 最優先で計算
(2 + 3) * 4
結果: 20
2
* / %
掛け算・割り算・余り
2 + 3 * 4
結果: 14
3
+ -
足し算・引き算
10 - 3 + 2
結果: 9
4
< > <= >=
大小比較
5 + 3 > 6
結果: true
5
== !=
等価比較
2 * 3 == 6
結果: true
❌ 間違った理解
式: 2 + 3 * 4

左から順番に計算すると思って...
2 + 3 = 5
5 * 4 = 20
間違った結果: 20
✅ 正しい理解
式: 2 + 3 * 4

優先順位に従って計算
3 * 4 = 12 （掛け算が先）
2 + 12 = 14 （その後足し算）
正しい結果: 14
🧠 実際のJavaプログラムで確認
int result1 = 2 + 3 * 4; // 結果: 14
int result2 = (2 + 3) * 4; // 結果: 20
boolean check = 5 + 3 > 6; // 結果: true (8 > 6)
boolean equal = 2 * 3 == 6; // 結果: true
💡 迷ったときの解決策
括弧 ( ) を積極的に使いましょう！

優先順位を覚えるより、括弧で明確にする方が安全です。

例：
• a + b * c → a + (b * c)
• (a + b) * c → 足し算を先にしたい場合
• (score >= 60) && (age >= 18) → 比較を明確に

括弧は何重にしても大丈夫です！
読みやすさと正確性が最優先。
