📚 Week 1: CSS基礎学習
/* 静的なCSS学習 */
.card {
  background: #f8f9fa;
  padding: 20px;
  border-radius: 8px;
  transition: all 0.3s ease;
}

.card:hover {
  transform: translateY(-5px);
  box-shadow: 0 8px 25px rgba(0,0,0,0.2);
}
🌟 Week 8.9: Thymeleaf統合
<!-- 動的CSS + Thymeleaf -->
<div class="card"
     th:classappend="${student.gpa >= 90} ? 'excellent' : 'standard'">
  <h3>[[${student.name}]]</h3>
  <p>成績: [[${student.gpa}]]</p>
</div>

/* データ値による動的スタイル */
.excellent {
  border-left: 5px solid #28a745;
  background: linear-gradient(135deg, #d4edda, #c3e6cb);
}
