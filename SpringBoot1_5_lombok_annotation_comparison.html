public class Student {
    private Long studentId;
    private String name;
    private String email;
    private Integer age;
    private LocalDate enrollmentDate;

    // デフォルトコンストラクタ
    public Student() {}

    // 全パラメータコンストラクタ
    public Student(Long studentId, String name,
                         String email, Integer age,
                         LocalDate enrollmentDate) {
        this.studentId = studentId;
        this.name = name;
        this.email = email;
        this.age = age;
        this.enrollmentDate = enrollmentDate;
    }

    // 5つのgetter（省略表示）
    public Long getStudentId() {
        return studentId;
    }
    // ... 他4つのgetter ...

    // 5つのsetter（省略表示）
    public void setStudentId(Long studentId) {
        this.studentId = studentId;
    }
    // ... 他4つのsetter ...

    // equals, hashCode, toString...
}

Lombok適用

Lombok使用後
@Entity
@Table(name = "students")
@Data
@NoArgsConstructor
@AllArgsConstructor
public class Student {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long studentId;

    @Column(nullable = false, length = 50)
    private String name;

    @Column(unique = true)
    private String email;

    private Integer age;

    @Column(name = "enrollment_date")
    private LocalDate enrollmentDate;
}
🎯 自動生成される内容:
• 全フィールドのgetter/setter
• デフォルトコンストラクタ
• 全パラメータコンストラクタ
• equals(), hashCode(), toString()

@Builder
Builderパターンを自動実装。可読性の高いオブジェクト作成が可能
