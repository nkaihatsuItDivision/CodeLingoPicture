✨ フォーカス・hover効果でユーザー体験を向上

ユーザーの操作に対する視覚的フィードバックが使いやすさを大きく左右します

🖱️ hover効果（マウスを乗せた時）
CSSコード
.form-input { width: 100%; padding: 12px; border: 2px solid #ddd; border-radius: 6px; font-size: 16px; transition: all 0.3s ease; } .form-input:hover { border-color: #17a2b8; background-color: #f0f8ff; transform: scale(1.02); }
効果：
• 枠線の色が変わる
• 背景色が薄い青に
• わずかに拡大してアクティブ感を演出
実際にマウスを乗せてみてください
👆 入力欄にマウスカーソルを乗せてください
状態：通常
🎯 focus効果（クリック・入力時）
CSSコード
.form-input:focus { outline: none; border-color: #28a745; box-shadow: 0 0 0 3px rgba(40, 167, 69, 0.25); background-color: #f8fff8; }
効果：
• ブラウザ標準の青い枠線を削除
• 緑色の枠線に変更
• 光る効果（box-shadow）を追加
• 背景色を薄い緑に
クリックして入力してみてください
👆 入力欄をクリックして文字を入力してください
状態：通常
🎨 hover + focus の組み合わせ
完全なCSSコード
.form-input { width: 100%; padding: 12px; border: 2px solid #ddd; border-radius: 6px; font-size: 16px; font-family: inherit; background-color: white; box-sizing: border-box; transition: all 0.3s ease; } /* hover状態 */ .form-input:hover { border-color: #999; background-color: #f9f9f9; } /* focus状態（hoverより優先） */ .form-input:focus { outline: none; border-color: #6f42c1; box-shadow: 0 0 0 3px rgba(111, 66, 193, 0.25); background-color: #faf8ff; }
実用的なフォーム例
👆 マウスを乗せたり、クリックして入力してみてください
