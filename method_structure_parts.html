メソッドの構造と部品を理解しよう

メソッドの基本構文
public static int calculateSum(int a, int b) { }
修飾子
public static
現在は「おまじない」として覚えてOK
※ Week 3で詳しく学習予定
戻り値の型
メソッドが返す値の種類：
• void → 何も返さない
• int → 整数を返す
• String → 文字列を返す
• boolean → true/falseを返す
メソッド名
処理内容が分かる名前をつける：
• calculate → 計算する
• show → 表示する
• check → 判定する
• judge → 評価する
引数（パラメータ）
メソッドに渡すデータ：
• int a, int b → 2つの整数
• String name → 1つの文字列
• () → 引数なし
複数の場合はカンマで区切る
void型（戻り値なし）
public static void greetUser(String name) {
    System.out.println("こんにちは、" + name + "！");
    System.out.println("頑張って学習しましょう！");
    // returnは書かない
}
使い方：
greetUser("田中さん");
greetUser("佐藤さん");

結果： 画面に文字が表示される
戻り値あり型
public static int calculateSum(int a, int b) {
    int sum = a + b;
    return sum; // 結果を返す
}

public static boolean isEven(int num) {
    return num % 2 == 0;
}
使い方：
int result = calculateSum(10, 20);
boolean even = isEven(5);

結果： 変数で受け取って使用
