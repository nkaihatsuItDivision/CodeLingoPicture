📚 フォーム要素まとめ - インタラクティブ学習

学習した要素を実際に触って復習しよう！
各要素をクリックして動作を確認し、知識を定着させましょう
カテゴリで絞り込み:
🔒
パスワード入力
type="password"
✨ 特徴:
  • 入力文字が●や•で隠される
  • セキュリティを保護
  • 他人に見られる心配なし
🎯 使用場面:
パスワード、PIN番号、秘密の質問の答えなど
🔢
数値入力
type="number"
✨ 特徴:
  • 数字以外の入力を制限
  • スマホで数字キーボード表示
  • min/max属性で範囲指定可
🎯 使用場面:
年齢、個数、価格、評点など
📅
日付入力
type="date"
✨ 特徴:
  • カレンダー表示で選択
  • YYYY-MM-DD形式で統一
  • 無効な日付を自動防止
🎯 使用場面:
生年月日、予約日、開始日・終了日など
☑️
チェックボックス
type="checkbox"
好きなものを選んでください（複数選択可）:
✨ 特徴:
  • 複数選択可能
  • 0個〜全部まで選択OK
  • 独立した選択肢
🎯 使用場面:
趣味、興味分野、利用規約同意、オプション選択など
🔘
ラジオボタン
type="radio"
性別を選択してください（1つだけ）:
✨ 特徴:
  • 1つだけ選択可能
  • 同じname属性でグループ化
  • 相互排他的な選択肢
🎯 使用場面:
性別、支払い方法、満足度、商品サイズなど
📞
電話番号入力
type="tel"
✨ 特徴:
  • スマホで数字キーボード表示
  • ハイフンも入力可能
  • 通話機能との連携
🎯 使用場面:
電話番号、FAX番号、緊急連絡先など
📝
複数行テキスト
textarea
✨ 特徴:
  • 複数行のテキスト入力
  • サイズ調整可能
  • 改行を含む文章対応
🎯 使用場面:
お問い合わせ内容、感想、コメント、説明文など
📋
プルダウン選択
select
✨ 特徴:
  • 省スペースで多くの選択肢
  • 1つだけ選択可能
  • 決まった選択肢から選ぶ
🎯 使用場面:
都道府県、職業、年代など選択肢が多い場合
🏷️
ラベル
label
✨ 特徴:
  • クリック範囲を拡大
  • アクセシビリティ向上
  • ユーザビリティ改善
🎯 使用場面:
すべてのフォーム要素に必須
🎯 要素選択の判断基準
機密情報ですか？
YES ↓
type="password"
数値のみですか？
YES ↓
type="number"
日付ですか？
YES ↓
type="date"
複数選択？
YES ↓
type="checkbox"
選択肢が2-4個？
YES ↓
type="radio"
選択肢が5個以上？
YES ↓
select
