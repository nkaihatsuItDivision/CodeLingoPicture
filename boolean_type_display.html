✅❌ boolean型の結果表示
🤔 boolean型とは？
比較演算子の結果を保存する特別な型です。
「正しい（true）」か「間違い（false）」の2つの値だけを持てます。
true
正しい・真
false
間違い・偽
// boolean型の変数宣言と使用例 boolean result = (10 > 3); System.out.println("10 > 3 は " + result); boolean isEqual = (5 == 5); System.out.println("5 == 5 は " + isEqual); boolean isDifferent = (8 != 8); System.out.println("8 != 8 は " + isDifferent);
🖥️ 実行結果
10 > 3 は true
5 == 5 は true
8 != 8 は false
🎯 年齢制限チェック
int age = 18; boolean canVote = (age >= 18); System.out.println("選挙権: " + canVote); // 結果: 選挙権: true
📊 成績判定
int score = 75; boolean isPass = (score >= 60); boolean isExcellent = (score >= 80); System.out.println("合格: " + isPass); System.out.println("優秀: " + isExcellent); // 結果: 合格: true // 結果: 優秀: false
💰 予算チェック
int money = 1500; int itemPrice = 2000; boolean canBuy = (money >= itemPrice); System.out.println("購入可能: " + canBuy); // 結果: 購入可能: false
🔢 偶数・奇数判定
int number = 7; boolean isEven = (number % 2 == 0); System.out.println("偶数: " + isEven); // 結果: 偶数: false // （7は奇数なので）
🎯 重要なポイント
• boolean型は true と false の2つの値だけを持てます
• 比較演算子（==, !=, >, <, >=, <=）の結果は必ずboolean型になります
• true/falseは そのまま文字列と連結 して表示できます
• 次回学習するif文（条件分岐）で大活躍します！
• 日常の判断（「合格？」「購入可能？」）をプログラムで表現できます
