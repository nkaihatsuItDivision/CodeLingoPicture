プログラムの実行の流れ

mainメソッドの実行
1
System.out.println("プログラム開始");
2
int result = calculateSum(10, 20);
→
4
System.out.println("合計: " + result);
5
System.out.println("プログラム終了");
←
calculateSumメソッド
public static int calculateSum(int a, int b) {
    int sum = a + b;
    return sum;
}
3
戻り値: 30

📋 実行の流れの詳細説明
1
プログラム開始：
Javaプログラムが実行されると、必ずmainメソッドから開始されます
2
メソッド呼び出し：
mainメソッドからcalculateSum(10, 20)を呼び出し、引数（10と20）を渡します
3
メソッド実行と戻り値：
calculateSumメソッドで計算を実行し、結果（30）をreturnでmainメソッドに返します
4
結果を受け取り：
mainメソッドに戻り、戻り値をresult変数で受け取って処理を続行します
5
プログラム終了：
mainメソッドの最後まで実行されると、プログラムが終了します
