ArrayList基本操作の視覚的解説

動作を見ながらArrayListの4つの基本操作を理解しましょう

➕
add() - 要素の追加
ArrayListの末尾または指定位置に要素を追加
// 末尾に追加 ArrayList<String> fruits = new ArrayList<>(); fruits.add("りんご"); fruits.add("バナナ"); // 特定位置に挿入 fruits.add(1, "いちご");
サイズ: 0
1
add()メソッドを呼び出し
2
必要に応じて内部配列を拡張
3
要素を追加し、サイズを更新
👁️
get() - 要素の取得
指定したインデックスの要素を取得
// インデックス指定で取得 String firstFruit = fruits.get(0); String secondFruit = fruits.get(1); // 取得した要素を使用 System.out.println(firstFruit);

要素を返す（O(1)の高速アクセス）
✏️
set() - 要素の変更
指定したインデックスの要素を新しい値に変更
// 指定位置の要素を変更 fruits.set(1, "ぶどう"); // 変更前の値を取得 String oldValue = fruits.set(2, "みかん");
