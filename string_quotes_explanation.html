String型の正しい書き方
📝 重要なルール
文字列は必ず
" " (ダブルクォート)
で囲む！
"
✅ ダブルクォート
String型で使用
VS
'
❌ シングルクォート
Javaでは使えない
✅ 正しい書き方
String name = "田中太郎";
日本語もOK
String city = "東京";
短い文字列
String message = "こんにちは、世界！";
記号も含める
String email = "tanaka@example.com";
英数字と記号
String empty = "";
空の文字列
❌ エラーになる書き方
String name = 田中太郎;
クォートが無い
String city = '東京';
シングルクォート
String message = "こんにちは';
クォートが混在
String text = "未完成;
閉じクォートが無い
String number = 123;
数字だがクォート無し
🔍 特別なケース
数字も文字列にできる：
String ageText = "25"; // 文字列として扱う int age = 25; // 数値として扱う
「"25"」と「25」は全く別のもの！
文字列の中にクォートを入れる：
String quote = "彼は「こんにちは」と言った";
日本語の「」なら問題なし
🎯 覚えておくべきポイント
1. String型は必ず " " (ダブルクォート) で囲む
2. ' ' (シングルクォート) は使えない
3. 数字でも " " で囲めば文字列になる
4. 空の文字列は "" と書く
