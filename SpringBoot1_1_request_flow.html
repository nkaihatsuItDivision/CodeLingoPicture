🔄 Spring Boot リクエスト処理フロー

ブラウザからWebページ表示までの処理の流れ

1
🌐 ブラウザからリクエスト送信
ユーザーがブラウザでURLにアクセスします
http://localhost:8080/
// ユーザーがブラウザのアドレスバーにURLを入力 // または、リンクをクリック
2
🎮 Controller が自動で呼び出される
Spring Bootが自動でマッピングされたメソッドを実行
@Controller
public class HomeController {
@GetMapping("/") // このアノテーションで自動マッピング
public String home() {
return "index"; // templates/index.html を指定
}
}
3
🖼️ Thymeleaf が HTML を生成
return "index" により templates/index.html が処理される
// templates/index.html
<!DOCTYPE html>
<html lang="ja">
<head>
<title>学生管理システム</title>
</head>
<body>
<h1>🎉 Spring Boot学習開始！</h1>
</body>
</html>

4
📱 ブラウザに HTML が表示される
生成されたHTMLがブラウザに送信され、美しいWebページとして表示
// ブラウザでの表示結果
🎉 Spring Boot学習開始！
