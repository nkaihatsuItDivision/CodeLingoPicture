実践演習 - 高機能学生クラスの設計概要
🎓 Student クラス設計
🔒 privateフィールド（カプセル化）
private String name; // 学生名
private int age; // 年齢
private double gpa; // GPA
private List<String> subjects; // 履修科目
private Date enrollmentDate; // 入学日
⚙️ getter/setterメソッド（制御されたアクセス）
public String getName() / setName(String) // 検証付き
public int getAge() / setAge(int) // 範囲チェック
public double getGpa() / setGpa(double) // 範囲チェック
public List<String> getSubjects() / setSubjects(List) // 防御的コピー
public Date getEnrollmentDate() / setEnrollmentDate(Date) // 防御的コピー
📊 読み取り専用プロパティ（計算結果）
public String getFullInfo() // 完全情報の文字列表現
public int getSubjectCount() // 履修科目数
public int getEnrollmentYears() // 入学からの年数
🛡️ カスタムバリデーション
setterでデータの妥当性を検証
setName(String name) {
  if (name == null || name.trim().isEmpty()) {
    throw new IllegalArgumentException("名前は必須");
  }
  this.name = name.trim();
}
🔒 防御的コピー
コレクションの外部変更を防止
getSubjects() {
  return new ArrayList<>(this.subjects);
}
setSubjects(List<String> subjects) {
  this.subjects = new ArrayList<>(subjects);
}
📊 読み取り専用プロパティ
計算結果や統計情報の提供
getFullInfo() {
  return String.format("名前: %s, 年齢: %d, GPA: %.2f",
                    name, age, gpa);
}
🔐 セキュリティ保護
不正なアクセスや変更を完全に防止
// ❌ 直接アクセス不可
// student.name = ""; // コンパイルエラー

// ✅ 制御されたアクセスのみ
student.setName("田中太郎"); // 検証通過
