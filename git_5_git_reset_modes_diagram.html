Git Reset の3つのモード
--soft
git reset --soft HEAD~1
✅ コミットのみ取り消し
✅ ファイル内容は保持
✅ ステージング状態も保持
✅ すぐに再コミット可能
最も安全
--mixed
git reset HEAD~1
⚠️ コミットを取り消し
⚠️ ステージングも解除
✅ ファイル内容は保持
⚠️ git add が必要
注意が必要
--hard
git reset --hard HEAD~1
❌ コミットを取り消し
❌ ステージングも解除
❌ ファイル内容も削除
❌ 変更が完全に消去
