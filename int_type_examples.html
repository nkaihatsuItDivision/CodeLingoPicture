int型で扱える数値の種類
📊 実際の使用例
👤 年齢
int age = 25;
✨ 人の年齢、学年など
📚 テストの点数
int score = 85;
✨ 点数、評価、ランキング
🔢 個数・カウント
int count = 100;
✨ 在庫数、アクセス数など
💰 価格（円）
int price = 1500;
✨ 商品価格、料金など
➕➖ 正の数・負の数・ゼロ
正の数
int positive = 100; int height = 175;
身長、体重、距離など
負の数
int temperature = -5; int balance = -1000;
気温、収支、高度差など
ゼロ
int zero = 0; int startCount = 0;
初期値、カウンターなど
📏 int型で扱える数値の範囲
-2,147,483,648 ～ 2,147,483,647
約-21億 ～ 約+21億（普通の用途では十分！）
⚠️ int型では扱えないもの
✅ 正しい
int score = 85; int price = 150; int count = 0;
整数はすべてOK！
❌ エラー
int average = 3.5; int pi = 3.14; int rate = 0.95;
小数点は使えません！
🎯 まとめ
int型 = 整数専用の変数
年齢、点数、個数、価格など「数えられる数」に最適！
