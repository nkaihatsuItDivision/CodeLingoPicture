📤 git push（ローカル → リモート）
  • 自分の変更を世界に公開
  • コミット後に実行
  • 他の人に共有したいとき
  • ポートフォリオ更新
  • バックアップ効果も
📥 git pull（リモート → ローカル）
  • 他の人の変更を取得
  • 作業開始時に実行
  • 最新状態にしたいとき
  • GitHub Web編集の反映
  • チーム開発で必須
